## 🎓 Projetos Acadêmicos em Destaque

### 🌻 Seguidor Solar de 2 Eixos - Estilo Girassol
Sistema automatizado de rastreamento solar que simula o comportamento de um girassol, seguindo a trajetória do sol em dois eixos para maximizar a captação de energia solar.
- **Tecnologias:** ESP32, Sensores LDR, Servomotores
- **Funcionalidades:** Rastreamento duplo eixo, automação solar

### 🌾 Sistema QualiSoja
Plataforma web para análise e classificação da qualidade de grãos de soja, integrando algoritmos de processamento de dados e banco de dados robusto.
- **Tecnologias:** Web Application, Banco de Dados
- **Área:** Agronegócio, Análise de Qualidade

### ⚡ Placa no Esyeda
Desenvolvimento de hardware personalizado para sistemas embarcados, focando em aplicações IoT e automação.
- **Tecnologias:** PCB Design, Microcontroladores
- **Funcionalidades:** Hardware Custom, IoT

### 🤖 Projetos de Inteligência Artificial
- **IA com Java:** Algoritmos de Machine Learning para análise preditiva
- **IA com Python:** Redes neurais, processamento de linguagem natural e visão computacional

### 🌐 Trabalho de Redes - VPN com GNS3
Projeto de redes utilizando o simulador GNS3 para implementação e teste de infraestrutura VPN, incluindo configuração de túneis seguros e políticas de segurança.
