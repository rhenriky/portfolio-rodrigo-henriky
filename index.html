Sobre Mim

Engenheiro da Computação formado pelo Instituto Federal de Mato Grosso (IFMT), uma das principais instituições de ensino técnico e superior do estado. O IFMT é reconhecido pela excelência acadêmica e pela formação de profissionais altamente qualificados em tecnologia. Durante minha graduação, tive acesso a laboratórios modernos, professores experientes e uma metodologia de ensino que integra teoria e prática de forma excepcional.

Minha jornada acadêmica no IFMT me permitiu trabalhar em projetos diversificados e desafiadores, desde sistemas web para análise de dados agrícolas até dispositivos IoT para automação solar. A instituição proporcionou um ambiente de aprendizado rico em recursos tecnológicos, incluindo laboratórios de redes, sistemas embarcados e desenvolvimento de software. Tenho experiência sólida no desenvolvimento de soluções que integram software, hardware e sistemas embarcados, com especialização em Inteligência Artificial utilizando Java e Python.

Principais Projetos Acadêmicos:

  • Sistema QualiSoja: Plataforma web para análise e classificação de qualidade de grãos de soja, integrando banco de dados e algoritmos de processamento.
  • Placa no Esyeda: Desenvolvimento de hardware personalizado para sistemas embarcados, com foco em IoT e automação.
  • Seguidor Solar de 2 Eixos: Sistema automatizado tipo girassol para rastreamento solar, maximizando a eficiência energética através de sensores e motores de precisão.
  • IA com Java: Implementação de algoritmos de Machine Learning e processamento de dados utilizando Java para análise preditiva e classificação.
  • IA com Python: Desenvolvimento de modelos de Inteligência Artificial com Python, incluindo redes neurais, processamento de linguagem natural e visão computacional.

Trabalho de Redes - VPN com GNS3

Projeto de redes desenvolvido utilizando o simulador GNS3 (Graphical Network Simulator) para implementação e teste de infraestrutura VPN (Virtual Private Network). O GNS3 permitiu simular topologias complexas de rede, configurar roteadores, switches e firewalls virtuais, além de testar cenários reais de conectividade segura entre diferentes localidades. Foram implementados túneis VPN IPSec, configurações de criptografia avançada e políticas de segurança para acesso remoto.

GNS3 VPN IPSec Roteadores Redes Segurança
